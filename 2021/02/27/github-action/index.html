使用 Github Action 為 CCRegionSelector 加上測試的 CI

  • github

首先先在 project 目錄下的 .github/workflows 資料夾中加上 yml 檔，我這邊命名為 CI-iOS.yml。
Github Action 可以設定的參數非常多，這邊目標只是在當 code push 到 master 時才跑測試確認沒問題。
